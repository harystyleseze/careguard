# x402 setup

CareGuard uses x402 to charge for certain API routes (for example, pharmacy comparison).

## Required environment variables

- `STELLAR_NETWORK`
  - `testnet` (default): `OZ_FACILITATOR_API_KEY` is optional, but protected x402 routes will return an error until configured.
  - `public`: `OZ_FACILITATOR_API_KEY` is required at boot.
- `OZ_FACILITATOR_API_KEY`
  - OpenZeppelin Channels facilitator API key used by the x402 middleware.
- `X402_FACILITATOR_URL` (optional)
  - Defaults to the testnet facilitator URL.

## Notes

- If `STELLAR_NETWORK=public` and `OZ_FACILITATOR_API_KEY` is missing, the server exits with a clear error message.
